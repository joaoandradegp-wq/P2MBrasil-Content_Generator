## 📌 Sobre

O **BBCode Layout Generator** é uma ferramenta desenvolvida em PHP para automatizar a criação de layouts completos de postagens em fóruns que utilizam **BBCode com suporte a HTML ([dohtml])**.

O sistema foi projetado para gerar, de forma rápida e padronizada, posts ricos em conteúdo, incluindo:

- Capa (poster)
- Screenshots
- Ficha técnica
- Sinopse
- Elenco
- Informações técnicas
- Links organizados por partes (CDs)
- Dados de upload

Tudo isso pronto para **copiar e colar diretamente no fórum**.

---

## ⚙️ Como funciona

- O usuário preenche um formulário com os dados do conteúdo  
- O sistema processa e valida as informações  
- Campos vazios são automaticamente tratados com valores padrão  
- O código final é gerado em formato **BBCode estruturado com HTML**  
- Basta copiar o conteúdo gerado e colar no fórum  

---

## 🧠 Funcionalidades

- Geração automática de layout completo em BBCode  
- Estrutura visual organizada com tabelas e imagens  
- Tratamento automático de campos vazios  
- Suporte a múltiplas imagens (poster + cenas)  
- Geração de ficha técnica detalhada  
- Inclusão de comentários do uploader  
- Organização de downloads por múltiplos CDs  

### 🔗 Suporte a múltiplas fontes de download

O sistema permite estruturar links de download separados por:

- **Peer2Mail (P2M)**  
- **Ilusions Downloader (ID)**  
- **Iddeias (IDS)**  

Cada um com:

- Conta  
- Login  
- Senha  

Organizados automaticamente por CD.

## 🎯 Quando usar

- Criação de posts padronizados em fóruns  
- Compartilhamento de mídia com layout estruturado  
- Organização de downloads em múltiplas partes  
- Padronização visual de conteúdo  

---

## 🏛️ Contexto

Este projeto reflete uma época onde fóruns utilizavam amplamente **BBCode avançado com HTML embutido**, exigindo alto nível de padronização manual para publicações.

A ferramenta foi criada para eliminar esse esforço, automatizando completamente o processo.
